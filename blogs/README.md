# LibreKAi Blog

Place your blog posts in this directory following the naming convention:
`YYYY-MM-DD_N.md` (e.g., `2026-01-01_1.md`, `2026-01-01_2.md` for multiple posts on the same day)

## Front Matter

Each blog post should start with YAML front matter:

```markdown
---
title: Post Title
author: your-username
author_icon: /path/to/avatar.jpg  # optional
tags: [tag1, tag2]
---

Your content here...
```

## Notes

- Blog posts are served at `/blogs/YYYY-MM-DD/N`
- The `author_icon` field is optional
- Tags can be used to filter posts
